# XEQUES RFC-0003
## Reference Execution Semantics

---

## 1. Required State

Each device MUST maintain:
- Device identifier
- Monotonic sequence counter
- Execution log

---

## 2. Reference Algorithm


---

## 3. Determinism

Identical inputs MUST produce identical receipts.

---

## 4. Failure Semantics

If execution fails:
- No receipt MUST be emitted
